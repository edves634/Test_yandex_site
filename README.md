### 1.2. [Анализ сетевой загрузки сайта в FireFox DevTools (Network)](https://github.com/edves634/Test_yandex_site/blob/main/Сеть/Анализ_загрузки.md)

**Краткое описание:**  
В отчёте представлены результаты анализа загрузки страницы с помощью панели Network в FireFox DevTools. Проверка выполнялась на разрешениях 375px, 768px, 1366px и 1920px с эмуляцией Regular 3G и отключённым кэшем. Выявлены критические проблемы: бесконечные запросы шрифта Font Awesome на 1920px (страница не загружается), отсутствие адаптивной загрузки изображений (на мобильных версиях передаётся до 2,76 МБ), избыточный вес PNG-файлов, большое количество блокирующих JS-скриптов и отсутствие кэширования. Даны рекомендации по оптимизации.

### 1.3. [Производительность (Lighthouse)](https://github.com/edves634/Test_yandex_site/blob/main/Lighthouse/Lighthouse.md)

Lighthouse для десктопа показал хорошие оценки (производительность 0.87, лучшие практики 0.96), но выявил проблемы с весом изображений, кэшированием и контрастностью. На мобильных устройствах производительность заметно ниже (0.69) — критическое время LCP и TTI превышает 7.8 с. Основная рекомендация: оптимизировать изображения, настроить кэширование и устранить блокирующую отрисовку CSS.

### 1.4. [Ошибки в вёрстке (несоответствие макету Figma)](https://github.com/edves634/Test_yandex_site/blob/main/Ошибки%20в%20верстке/Макет%20фигма_верстка.md)

В ходе проверки выявлены значительные расхождения между реализованной вёрсткой и макетом Figma на всех разрешениях (1366px, 375px, 1920px). 
Основные проблемы: отсутствие декоративных слоёв (винтажный стиль утерян), неверная структура секций, неполные бегущие строки, некорректные данные в карточках участников и избыточные JS-манипуляции с DOM.
